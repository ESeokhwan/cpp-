### `1941` 소문난 칠공주

- 백트랙킹과 bfs를 합쳐서 풀었다.
- 현 상태를 마킹하고 뒤로 돌아가도록 구현하는 것이 어려웠다.
- 아래 코드처럼 총 개수를 세어야 하기 때문에 중간에 조건 달성했다고 끝내면 안되고 끝까지 가야했다.

  ```
  // solution.cpp line 39

  // Correct
  if(cnt >= 7 && s_cnt >= 4) {
    ans += 1;
    return;
  }
  if(7 - cnt < 4 - s_cnt) return;
  if(cnt >= 7) return;

  // Wrong
  if(s_cnt >= 4) {
    ans += 1;
    return;
  }
  if(7 - cnt < 4 - s_cnt) return;
  if(cnt >= 7) return;
  ```
