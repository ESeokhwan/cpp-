### `2228` 구간 나누기

- 연속된 숫자의 합 중 최대값 찾는 알고리즘과 dp를 함께 사용하여 풀었다.
- dp의 초기값 설정이 빠져있어서 틀렸다.

  ```
  // solution.cpp line 37

  // Correct
  if(y == 1) {
    dp[x][y] = submax[0][x];
  } else if(x == 2 * (y - 1) + 1) {
    int tmp = 0;
    for(int i = 0; i < x; i++) {
      if(i % 2 == 0) tmp += nums[i];
    }
    dp[x][y] = tmp;
  }

  // Wrong
  if(x == 2 * (y - 1) + 1) {
    int tmp = 0;
    for(int i = 0; i < x; i++) {
      if(i % 2 == 0) tmp += nums[i];
    }
    dp[x][y] = tmp;
  }
  ```
