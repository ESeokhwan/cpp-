### `10800` 컬러볼

- 누적합 아이디어를 이용해 쉽게 풀 수 있는 문제였다.
- color이 0-base가 아니라 1-base라서 색상별 누적합 배열의 크기를 딱 200,000에 맞추면 틀리는 경우가 발생했다.

  ```
  // solution.cpp line 22

  // Wrong
  int acc_by_color[200000];

  // Correct
  int acc_by_color[200001];
  ```
