### `2169` 로봇 조종하기

- `i`, `j` / `n`, `m` 유심히 살펴보자 이거 때문에 1시간 날렸다.

  ```
  // solution.cpp line 25

  // Correct
  for(int j = 0; j < m; j++) {
    if(j == 0) dp[i][j] = _map[i][j];
    else dp[i][j] = dp[i][j - 1] + _map[i][j];
  }


  // Wrong
  for(int j = 0; j < n; j++) {
    if(j == 0) dp[i][j] = _map[i][j];
    else dp[i][j] = dp[i][j - 1] + _map[i][j];
  }

  ```
